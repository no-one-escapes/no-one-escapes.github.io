Базовый курс по наращиванию ресниц

«От классики до объема с нуля»

Качественное базовое обучение наращиванию ресниц- это крепкий фундамент для развития мастера. В DN Academy мы подготавливаем специалистов с нуля, обеспечивая их прочной основой для дальнейшей работы. Мастера, получая сертификат нашей школы, могут выполнять объемы от классического до тройного. У них не возникает сложностей при подборе индивидуального эффекта для клиента. Так же, мы обеспечиваем всех своих учеников правовой информацией, знаниями по маркетингу. Первые два года существования школы, мы специализировались на подготовке специалистов с нуля, поэтому знаем КАК правильно донести информацию и как выстроить пошаговую стратегию развития мастера-новичка. Наши лучшие выпускники, востребованные специалисты, открывшие свои школы и студии, начинали с этого курса!

ПРОГРАММА КУРСА

1 день. Теоретическая часть. Практика на манекене. Формирование объемов

ПОДГОТОВИТЕЛЬНЫЙ БЛОК

— Основы безопасного наращивания (противопоказания, меры предосторожности). Требования государственного стандарта.
— Дезинфекция и стерилизация материалов, по СанПИН;
— Подробный разбор материалов для наращивания ресниц, сравнение фирм и цен;
— Обустройство рабочего места мастера.

БЛОК МОДЕЛИРОВАНИЕ

— Создание различных эффектов в наращивании ресниц. Отрисовка;
— Как использовать индивидуальные особенности клиента при выборе эффекта наращивания;
— Правильная разметка с практической отработкой.

БЛОК НАРАЩИВАНИЕ РЕСНИЦ

— Этапы наращивания ресниц (видеопрезентация);
— Правильный демакияж;
— Различные способы изоляции;
— Постановка рук;
— Правильная посадка мастера;
— Работа с пинцетами, искусственной ресницей, клеем для наращивания ресниц;
— Постановка ресницы;
— Правила наращивания;
— Условия долгой носки.

ПРАКТИЧЕСКИЕ ЗАДАНИЯ на манекене
ТЕОРЕТИЧЕСКАЯ ЧАСТЬ ОБЪЕМНОГО НАРАЩИВАНИЯ

— Способы формирования пучков в различных техниках;
— Геометрия пучка;
— Особенности работы с клеем;
— Техника постановки пучков разных объемов.

БЛОК РАБОТЫ С КЛИЕНТАМИ

— Формирование цены на наращивание;
— Позиционирование мастера;
— Социальные сети и способы продвижения;
— Клиентоориентированность.

День 2. Практическая часть. Работа с моделями. Классическое наращивание.

День 3. Объемное наращивание ресниц. Углубленное моделирование.

ТЕОРЕТИЧЕСКИЙ БЛОК

— Работа с рядностью;
— Микс изгибов;
— Smart наращивание;
— Правильная укладка пучков;
— Ровная ресничная линия;
— Работа с асимметрией.

ПРАКТИЧЕСКИЙ БЛОК

— Отработка на моделях объемного наращивания 2-3 D

МАТЕРИАЛЫ

— Чек лист по необходимым материалам для начала работы;
— Чек лист по работе с клиентом (подбор эффекта);
— Чек лист по бесплатному продвижению Lash мастера;
— Карта клиента;
— Тетрадь ученика.

курс
